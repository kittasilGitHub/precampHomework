2.ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กําหนดให้

differentiation

tag differentiation
hr & br
  • hr : Define thematic changes in the content
  • br : Insert single line breaks in a text
ul & ol
  • ul : Defines an unordered HTML list
  • ol : Defines an ordered list. An ordered list can be numerical or alphabetical.
select & checkbox & radio button
  • select : Create a drop-down list
  • checkbox : Let the user select one or more options of a limited number of choices
  • radio: Radio buttons let a user select only one of a limited number of choices
div & span
  • div : defines a division or a section in an HTML document.
  • span : an inline container used to mark up a part of a text, or a part of a document
script & style
  • script : used to embed a client-side script (JavaScript).
  • style : used to define style information (CSS) for a document.
